# ABit

Keyboard centric organizer application

### Prerequisite

- [POSIX-compliant shell][shell]
- [git][git] >= 2
- [github-cli][github-cli] >= 2
- [volta][volta]
- [docker][docker]
- [docker-compose][docker-compose]

### Branch Organization

- `main`: latest stable release
- `dev`: latest development changes
